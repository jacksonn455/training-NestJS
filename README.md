NestJs
===============================================

*Pré-requisitos* <br>
- Certifique-se de que o NodeJs deve estar com a versão >= 12 no seu sistema operacional. <br>

## Instalar

```bash
#Global install
$ npm i -g @nestjs/cli

#Install all dependencies
$ npm install
```

## Versão

```bash
$ nest -v
```

## Criar projeto

```bash
#New Project
$ nest new project-name
```

## Rodar

```bash
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
